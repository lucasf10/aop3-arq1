# AOP3 – Consulta de Preços de Combustíveis

Projeto desenvolvido para a disciplina **Arquitetura de Dados Relacionais I**.

## Deploy
https://aop3arg1vercel.app


## Tecnologias
- Next.js
- TypeScript
- MySQL

## Como rodar

```
npm install
npm run dev
```

## Variáveis de ambiente
Crie um .env.local:
```
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
```
